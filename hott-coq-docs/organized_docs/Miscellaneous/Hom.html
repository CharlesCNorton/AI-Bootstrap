Quoting the MIT 18.705 Lecture Notes:
Let C and D be categories, F : C → D and G : D → C functors. We call (F, G) an adjoint pair, F the left adjoint of G, and G the right adjoint of F if, for each object A : C and object A' : D, there is a natural bijection
Hom_D (F A) A' ≅ Hom_C A (G A')
Here natural means that maps B → A and A' → B' induce a commutative diagram: 
       Hom_D (F A) A' ≅ Hom_C A (G A')
             |                 |
             |                 |
             |                 |
             |                 |
             V                 V
       Hom_D (F B) B' ≅ Hom_C B (G B')
We want to simpl out the notation machinery
  Local Opaque NaturalIsomorphism.

                     (Category.Prod.prod (Category.Dual.opposite C) D)

                  (@compose
                     (Category.Prod.prod (Category.Dual.opposite C) D)
